# 🎵 音乐播放器快速参考指南

## 🚀 快速开始

### 安装与部署
```bash
# 1. 安装 Wrangler
npm install -g wrangler

# 2. 配置 wrangler.toml
wrangler publish

# 3. 设置环境变量
wrangler secret put PASSWORD
```

### 基本使用
1. 打开应用 → 输入密码登录
2. 搜索歌曲 → 选择源和类型
3. 播放音乐 → 选择音质
4. 管理播放列表

## 🎯 核心功能一览

| 功能 | 快捷键 | 描述 |
|------|--------|------|
| 搜索 | Enter | 输入关键词并回车搜索 |
| 播放 | 左键单击 | 点击任何歌曲播放 |
| 暂停 | Space | 暂停/继续播放 |
| 下一首 | ⏭ | 播放下一首 |
| 上一首 | ⏮ | 播放上一首 |
| 播放模式 | 🔁 | 切换循环/随机播放 |
| 主题切换 | 🌙 | 暗黑/浅色主题 |

## 🎵 音质选择

选择播放音质：
- **128kbps** - 流量节省
- **192kbps** - 移动端推荐
- **320kbps** - 日常听歌 ⭐ 推荐
- **740kbps** - 专业用户
- **999** - 无损 (若源支持)

## ⌨️ 键盘快捷键

- `Enter` - 执行搜索
- `Space` - 播放/暂停
- `→` - 进度条前进10秒
- `←` - 进度条后退10秒
- `↑` - 增加音量
- `↓` - 减少音量

## 🐛 常见问题

### Q: 搜索无结果？
A: 
1. 检查网络连接
2. 尝试更换音乐源
3. 使用不同的搜索关键词
4. 检查浏览器控制台错误

### Q: 音乐无法播放？
A:
1. 确认已选择正确的音质
2. 尝试刷新页面
3. 检查浏览器是否支持 MP3
4. 尝试另一个音乐源

### Q: 图片无法加载？
A:
1. 这是正常的（某些源可能限制）
2. 应用会自动显示占位符
3. 不影响音乐播放

### Q: 怎样离线使用？
A:
1. PWA 支持（开发中）
2. 当前只支持在线播放
